Gadget Master Manager v6 Mobile PWA Edition

This version makes the web app installable like a mobile application.

What changed:
- iPhone/Android install support
- Apple mobile web app tags
- App manifest with icons
- Better mobile layout and safe-area support
- Improved service worker cache
- Same Supabase cloud connection

How to update GitHub:
1. Upload these extracted files to your GitHub repository.
2. Replace old index.html, app.js, style.css, manifest.json, and service-worker.js.
3. Upload the icons folder.
4. Commit changes.
5. Wait 1-2 minutes for GitHub Pages to redeploy.

How to install on iPhone:
1. Open the GitHub Pages app link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Tap Add.

How to install on Android:
1. Open the GitHub Pages app link in Chrome.
2. Tap the three-dot menu.
3. Tap Add to Home screen or Install app.
