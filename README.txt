Gadget Master Manager v5 Tickets + Invoices

New features:
- Repair ticket numbers
- Printable repair tickets
- Invoice numbers
- Printable sales invoices
- Customer phone on invoices
- IMEI / serial field on repairs
- Same Supabase project URL already included

Before using:
1. Open Supabase SQL Editor.
2. Run v5_upgrade.sql.
3. Open the app.
4. Paste your publishable key if needed.
5. Login with owner account.

Print:
- On Repairs tab, tap Print beside a ticket.
- On Sales / Invoices tab, tap Print beside an invoice.
